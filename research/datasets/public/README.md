# Datasets públicos

## Selecionado para o primeiro benchmark

**Keraal** — somente pesquisa, CC-BY-NC-SA.

Motivos: vídeos RGB anonimizados, pacientes e controles, anotações por profissional de reabilitação, BlazePose 33, OpenPose COCO, Kinect e Vicon.

Restrições:

- não usar para treinar modelo comercial;
- não redistribuir o conteúdo pelo repositório;
- manter atribuição e registro do download;
- resultados derivados continuam classificados como pesquisa;
- promoção comercial não pode depender deste dataset como conjunto de treinamento.

## Avaliados e não selecionados agora

- Fit3D: licença individual e não comercial, com proibição explícita de treinamento para uso comercial.
- UI-PRMD: valioso para esqueletos e exercícios corretos/incorretos, mas não atende sozinho ao benchmark RGB com os três estimadores.
- EC3D: relevante para correção 3D; termos do dataset precisam de verificação adicional antes da ingestão.
